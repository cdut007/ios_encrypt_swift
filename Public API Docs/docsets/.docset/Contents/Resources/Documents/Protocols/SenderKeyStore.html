SenderKeyStore

public protocol SenderKeyStore

The SenderKeyStoreprotocol must be adopted to provide the storage for sender keys.

  • store(senderKey:for:userRecord:)

    Store a serialized sender key record for a given (groupId + senderId + deviceId) tuple.

    Declaration

    Swift

    func store(senderKey: Data, for address: SignalSenderKeyName, userRecord: Data?) -> Bool

    Parameters

    senderKey

    The serialized record
    address

    the (groupId + senderId + deviceId) tuple
    userRecord

    Containing application specific data to be stored alongside the serialized record. If no such data exists, then this parameter will be nil.

    Return Value

    true on success, false on failure
