GroupCipher

public final class GroupCipher

The main entry point for Signal Protocol group encrypt/decrypt operations.

Once a session has been established with GroupSessionBuilder and a sender key distribution message has been distributed to each member of the group, this class can be used for all subsequent encrypt/decrypt operations within that session (i.e. until group membership changes).
