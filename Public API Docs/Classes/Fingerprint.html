Fingerprint

public class Fingerprint

Fingerprints can be used to compare identity keys across devices. A fingerprint consists of a human readable string of numbers and a data that can be transmitted to another device (e.g. through a QR Code).

    Construct a fingerprint (Version 0) generator for 60 digit numerics.

    Note

    The higher the iteration count, the higher the security level:
  • 1024 iterations ~ 109.7 bits
  • 1400 iterations > 110 bits
  • 5200 iterations > 112 bits
