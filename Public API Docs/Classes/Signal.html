    static func generateRegistrationId(extendedRange: Bool = false) throws -> UInt32

    Parameters

    extendedRange

    By default (false), the generated registration Id is sized to require the minimal possible protobuf encoding overhead. Specify true if the caller needs the full range of UInt32.max at the cost of slightly higher encoding overhead.

    Return Value

    The generated registration Id on success, or nil on failure

  • generatePreKeys(start:count:)

    Generate a list of PreKeys. Clients should do this at install time, and subsequently any time the list of PreKeys stored on the server runs low.

    Note

    Pre key IDs are shorts, so they will eventually be repeated. Clients should store pre keys in a circular buffer, so that they are repeated as infrequently as possible.
